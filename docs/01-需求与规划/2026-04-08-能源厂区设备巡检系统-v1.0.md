**版本**: v1.0  
**日期**: 2026-04-08  
**状态**: 设计阶段

---

## 项目概述

### 业务背景

能源厂区设备巡检系统是一套面向能源行业的设备巡检管理解决方案。系统主要解决厂区设备定时巡检、高危设备监控、巡检数据记录、异常整改管理等核心业务需求。

### 核心业务

- **定期巡检**: 厂区特定区域和高危设备需要定时巡检
- **数据采集**: 记录设备压力、流速等关键参数
- **异常处理**: 发现问题后提交整改，跟踪整改流程
- **轨迹追踪**: 记录巡检路线和实时位置，类似导航功能
- **数据分析**: 统计巡检完成率、设备异常率等指标

### 系统组成

- **PC 管理后台**: 管理人员使用，负责计划制定、数据查看、统计分析
- **移动端 APP**: 巡检员使用（H5），负责现场巡检执行、整改处理
- **数字大屏**: 监控中心使用，实时展示设备状态、巡检进度、视频监控

### 用户角色

- **系统管理员**: 系统配置、权限管理
- **业务主管**: 监督管理、审批决策
- **业务主办**: 制定巡检计划、维护路线和检查清单
- **巡检员**: 执行巡检任务、提交整改
- **整改人员**: 执行整改任务

### 整改流程

发现问题 → 提交整改 → 审核指派（按问题类型分配审核人）→ 执行整改 → 提交人复核 → 完成

## 功能范围

### PC 管理后台

1. **工作台**: 待办事项、统计图表、通知公告、制度文档
2. **数字大屏**: 设备状态、视频监控、巡检统计、实时轨迹

5. **整改管理**: 整改工单台账、统计分析（查看为主，操作在工作台待办）
6. **基础设置**: 巡检路线、检查清单、设备分类
7. **组织管理**: 部门管理、人员管理、岗位管理
8. **权限管理**: 角色管理、菜单管理
9. **系统配置**: 基础配置、问题类型、视频监控、通知公告、制度文档

### 移动端 APP

1. **工作台**: 待办事项、常用应用、通知公告、制度文档
2. **巡检模块**: 
   - 巡检任务列表
   - 导航模式执行巡检（类似地图导航）

3. **整改模块**: 发起整改、整改待办、整改记录
4. **我的**: 个人信息、系统设置、退出登录

## 系统架构设计

### 整体架构

```
┌─────────────────────────────────────────────────────────┐
│                      用户层                              │
├──────────────┬──────────────┬─────────────────────────┤
│  PC 管理后台  │  移动端 H5   │    数字大屏              │
│ Vue3 + AntD  │ Vue3 + Vant  │  Vue3 + ECharts         │
└──────────────┴──────────────┴─────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────┐
│                   应用服务层                             │
│  Spring Boot + MyBatis-Plus + Redis + WebSocket        │
├─────────────────────────────────────────────────────────┤
│  用户模块 │ 设备模块 │ 巡检模块 │ 整改模块 │ 统计模块   │
└─────────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────┐
│                   数据存储层                             │
│  MySQL（业务数据） + OSS（文件存储） + Redis（缓存）    │
└─────────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────┐
│                   外部集成                               │
│  视频监控 SDK（海康/大华） + 短信/邮件通知              │
└─────────────────────────────────────────────────────────┘
```

### 技术栈

**前端**
- PC 管理后台：Vue3 + Ant Design Vue + TypeScript + Vite
- 移动端：Vue3 + Vant + TypeScript + Vite（H5）
- 数字大屏：Vue3 + ECharts + DataV + WebSocket
- 状态管理：Pinia
- 路由：Vue Router 4

**后端**
- 框架：Spring Boot 3.x
- ORM：MyBatis-Plus
- 安全：Spring Security + JWT
- 缓存：Redis
- 实时通信：WebSocket（Spring WebSocket）
- 文档：Swagger/Knife4j

**数据库**
- 关系型：MySQL 8.0
- 缓存：Redis 7.x

**部署**
- 容器化：Docker + Docker Compose
- Web 服务器：Nginx
- 文件存储：阿里云 OSS / MinIO

**地图导航**
- 高德地图 Web API（PC 端）
- 高德地图移动端 SDK（H5）
- 支持路径规划、导航、轨迹绘制

## PC 管理后台功能详细设计

### 1. 工作台

**功能概述**
工作台是用户登录后的首页，提供个性化的工作视图和快速入口。

**核心功能**
- **欢迎信息**: 显示当前用户姓名、角色、登录时间
- **待办事项**: 
  - 待审核的整改工单（业务主管）
  - 待执行的整改任务（整改人员）
  - 待复核的整改工单（巡检员）
  - 超时未完成的巡检任务
  - 点击可直接跳转到对应详情页
- **常用应用**: 快捷入口卡片（可自定义）
  - 创建巡检计划
  - 创建临时巡检任务
  - 设备管理
  - 巡检统计
- **统计图表**: 
  - 本月巡检完成率（环形图）
  - 近 7 天巡检趋势（折线图）
  - 设备异常分布（柱状图）
  - 整改工单状态分布（饼图）
- **通知公告**: 最新 5 条公告，支持查看详情
- **制度文档**: 常用制度文档列表，支持在线预览和下载

### 2. 数字大屏

**功能概述**
实时监控中心大屏，用于监控室展示厂区整体运行状态。

**布局设计**
- 顶部：标题 + 当前时间 + 天气信息
- 左侧：设备状态统计、巡检统计
- 中间：厂区地图 + 实时巡检轨迹 + 视频监控网格
- 右侧：异常告警列表、整改工单统计

**核心功能**
- **设备状态监控**:
  - 设备总数、在线设备数、异常设备数
  - 设备分类统计（饼图）
  - 异常设备列表（实时滚动）
- **单设备详情**:
  - 点击设备可查看实时参数（压力、流速、温度等）
  - 参数超标时红色高亮显示
  - 显示该设备最近一次巡检时间和结果
- **视频监控**:
  - 支持接入海康威视、大华等主流监控设备
  - 网格布局展示多路视频（4/9/16 宫格可切换）
  - 点击视频可全屏查看
  - 支持云台控制（上下左右、焦距调节）
- **巡检实时监控**:
  - 地图上显示当前正在巡检的人员位置（实时更新）
  - 显示巡检路线和已完成的巡检点
  - 点击人员图标可查看当前任务详情
- **巡检统计**:
  - 今日巡检任务数、已完成数、进行中、超时数
  - 本月巡检完成率趋势图
  - 巡检员排行榜（按完成率）
- **异常告警**:
  - 设备参数超标告警（实时推送）
  - 巡检超时告警
  - 整改超期告警
  - 告警列表实时滚动，支持点击查看详情

### 3. 设备管理

**功能概述**
管理厂区所有设备的档案信息、分类、厂商信息和巡检历史。

**核心功能**
- **设备档案**:
  - 列表展示：设备编号、名称、分类、位置、状态、最近巡检时间
  - 支持搜索（设备编号、名称）、筛选（分类、状态、位置）
  - 新增/编辑设备：
    - 基本信息：编号、名称、型号、分类、位置（支持地图选点）
    - 技术参数：额定压力、额定流速、功率等（根据设备类型动态配置）
    - 厂商信息：选择厂商、购买日期、质保期
    - 附件上传：设备照片、说明书、合格证等
  - 设备详情：
    - 基本信息展示
    - 实时参数（如果设备支持物联网接入）
    - 巡检记录列表（时间、巡检员、结果、异常描述）
    - 整改记录列表
    - 操作日志
- **设备分类**:
  - 树形结构管理设备分类（如：压力设备 > 压力容器 > 储气罐）
  - 每个分类可配置检查清单模板
  - 支持拖拽调整分类层级
- **厂商信息**:
  - 厂商列表：名称、联系人、电话、地址
  - 关联设备数量统计
  - 新增/编辑厂商信息

### 4. 巡检管理

**功能概述**
制定巡检计划、生成巡检任务、跟踪巡检执行情况。

**核心功能**
- **巡检计划管理**:
  - 列表展示：计划名称、巡检路线、执行周期、负责人、状态
  - 新增/编辑计划：
    - 基本信息：计划名称、描述、负责人
    - 巡检路线：选择已配置的巡检路线
    - 执行周期：每日/每周/每月/自定义（支持 Cron 表达式）
    - 执行时间：具体时间段（如 08:00-10:00）
    - 超时设置：允许超时多久后仍可执行（如超时 2 小时内允许巡检）
    - 指派人员：选择巡检员（支持多人轮班）
  - 计划启用/停用
  - 查看计划生成的任务列表
- **巡检任务管理**:
  - 任务来源：
    - 定期任务：由巡检计划自动生成
    - 临时任务：手动创建（如临时抽查、应急巡检）
  - 列表展示：任务编号、计划名称、巡检路线、执行人、计划时间、状态
  - 任务状态：待执行、进行中、已完成、已超时、已取消
  - 筛选条件：时间范围、执行人、状态、路线
  - 任务详情：
    - 基本信息：任务编号、计划名称、路线、执行人、时间
    - 巡检路线地图展示（显示所有巡检点）
    - 巡检点完成情况列表：
      - 巡检点名称、签到时间、检查结果、异常描述、照片
      - 未完成的巡检点标记为红色
    - 巡检轨迹回放（地图上显示实际行走路线）
    - 整改工单列表（如果有发现问题）
  - 手动创建临时任务：
    - 选择巡检路线
    - 指派执行人
    - 设置执行时间
    - 备注说明
  - 任务取消（仅限未开始的任务）

### 5. 整改管理

**功能概述**
管理巡检中发现的问题整改流程，跟踪整改进度。

**整改流程**
发现问题 → 提交整改 → 审核指派 → 执行整改 → 提交人复核 → 完成

**核心功能**
- **整改工单台账**:
  - 列表展示：工单编号、问题描述、问题类型、严重程度、提交人、当前状态、提交时间
  - 筛选条件：时间范围、问题类型、严重程度、状态、提交人、整改人
  - 工单状态：待审核、待整改、整改中、待复核、已完成、已关闭
  - 工单详情：
    - 基本信息：工单编号、问题描述、问题类型、严重程度
    - 发现信息：发现时间、发现人、关联巡检任务、关联设备、位置
    - 现场照片、语音描述
    - 审核信息：审核人、审核时间、审核意见、指派整改人、要求完成时间
    - 整改信息：整改人、整改方案、整改过程照片、完成时间
    - 复核信息：复核人、复核时间、复核结果、复核意见
    - 流程日志：记录每个环节的操作时间和操作人
  - 导出功能：支持导出 Excel
- **统计分析**:
  - 整改工单统计：
    - 总数、待处理数、已完成数、超期数
    - 按问题类型统计（饼图）
    - 按严重程度统计（柱状图）
    - 按提交人统计（排行榜）
  - 整改效率分析：
    - 平均整改时长
    - 超期率
    - 一次复核通过率
  - 趋势分析：
    - 近 30 天整改工单趋势（折线图）
    - 问题高发设备/区域（热力图）

**操作说明**
- 整改工单的具体操作（审核、指派、执行、复核）在工作台的待办事项中处理
- 本模块主要用于查看工单台账和统计分析

### 6. 基础设置

**功能概述**
配置巡检路线、检查清单、设备分类等基础数据。

**核心功能**
- **巡检路线设置**:
  - 列表展示：路线名称、巡检点数量、预计耗时、创建人、创建时间
  - 新增/编辑路线：
    - 基本信息：路线名称、描述、预计耗时
    - 巡检点配置：
      - 添加巡检点：选择区域或设备
      - 设置巡检顺序（支持拖拽排序）
      - 每个巡检点绑定检查清单
      - 设置签到方式：二维码/NFC/GPS 范围
      - 设置停留时长要求（如至少停留 5 分钟）
    - 地图展示：
      - 在地图上标注所有巡检点位置
      - 显示推荐路线（自动规划最优路径）
      - 支持手动调整路线顺序
  - 路线预览：查看路线地图和巡检点列表
  - 路线复制：基于现有路线创建新路线
- **检查清单管理**:
  - 清单列表：清单名称、适用设备类型、检查项数量
  - 新增/编辑清单：
    - 基本信息：清单名称、适用设备类型、描述
    - 检查项配置：
      - 检查项名称（如"压力表读数"）
      - 检查方式：选择项/数值输入/文本描述/拍照
      - 正常范围（如压力：0-10 MPa）
      - 是否必填
      - 排序
    - 支持从模板导入检查项
  - 清单预览：查看完整的检查项列表
- **设备分类维护**:
  - 树形结构展示设备分类
  - 新增/编辑/删除分类
  - 为每个分类配置默认检查清单
  - 支持拖拽调整分类层级和顺序

### 7. 组织管理

**功能概述**
管理企业组织架构、部门信息、人员信息和岗位配置。

**核心功能**
- **部门管理**:
  - 树形结构展示部门层级
  - 新增/编辑/删除部门：
    - 部门名称
    - 上级部门
    - 部门负责人
    - 联系电话
    - 部门描述
  - 支持拖拽调整部门层级和顺序
  - 查看部门下的人员列表
- **人员管理**:
  - 左侧：部门树形结构
  - 右侧：选中部门的人员列表
  - 列表展示：姓名、工号、岗位、手机号、状态
  - 新增/编辑人员：
    - 基本信息：姓名、工号、性别、手机号、邮箱
    - 组织信息：所属部门、岗位
    - 账号信息：用户名、初始密码、角色
    - 状态：在职/离职
  - 人员调动：更换部门或岗位
  - 批量导入：支持 Excel 批量导入人员
  - 人员禁用/启用
- **岗位管理**:
  - 列表展示：岗位名称、所属部门、人员数量、描述
  - 新增/编辑/删除岗位：
    - 岗位名称
    - 所属部门
    - 岗位职责描述
  - 查看岗位下的人员列表

### 8. 权限管理

**功能概述**
管理系统角色和菜单权限，实现基于角色的访问控制（RBAC）。

**核心功能**
- **角色管理**:
  1. 列表展示：表格列包括 ID、名称、备注、授权人数、排序、创建时间、状态，支持角色名称搜索、状态筛选（启用/禁用）、分页显示
  2. 新增角色：填写名称、备注、排序（数值越小越靠前）
  3. 编辑角色：修改角色基本信息
  4. 删除角色：删除前需确认
  5. 分配权限：树形结构勾选可访问的菜单，支持展开/折叠、全选/不全选
  6. 状态切换：通过开关快速启用/禁用角色
  7. 导入/导出：批量导入角色数据、导出角色列表
  8. 预置角色（初始化数据）：系统管理员（所有权限）、巡检专员（日常巡检与维护）、维修工程师（故障诊断与修复）、备件管理员（备品备件管理）
- **菜单管理**:
  - 树形结构展示菜单层级
  - 菜单类型：目录/菜单/按钮
  - 新增/编辑/删除菜单：
    - 基本信息：菜单名称、菜单类型、图标、排序
    - 路由信息：路由地址、组件路径
    - 权限标识：用于按钮级权限控制（如 system:user:add）
    - 显示状态：显示/隐藏
  - 支持拖拽调整菜单顺序

### 9. 系统配置

**功能概述**
配置系统运行所需的各类参数和基础数据。

**核心功能**
- **基础配置**:
  - 系统名称、Logo
  - 登录页背景图
  - 系统版权信息
  - 文件上传限制（大小、格式）
  - 地图配置：高德地图 API Key
- **问题类型配置**:
  - 列表展示：问题类型名称、审核人、排序
  - 新增/编辑问题类型：
    - 类型名称（如：设备故障、安全隐患、环境问题）
    - 指定审核人（该类型问题提交后由谁审核）
    - 排序
  - 问题类型启用/禁用

## 边界情况处理

### 巡检任务执行

**超时处理**
- 任务超过计划结束时间后，根据计划配置的"允许超时时长"判断是否还能执行
- 超过允许时长后，任务状态变为"已超时"，不允许继续执行
- 超时任务需要业务主办手动处理（重新指派或取消
**离线巡检**
- 移动端支持离线模式，巡检数据暂存本地
- 恢复网络后自动上传数据
- 离线期间无法查看实时视频监控

**GPS 信号弱**
- 室内或信号弱区域，GPS 定位可能不准确
- 优先使用二维码/NFC 签到方式
- GPS 范围签到作为辅助手段

**巡检点签到失败**
- 二维码损坏：支持手动输入巡检点编号
- NFC 标签失效：切换到二维码或 GPS 签到
- GPS 范围外：提示距离巡检点的距离，引导前往

**中途退出**
- 巡检过程中退出 APP，数据自动保存
- 再次进入时可继续未完成的任务
- 已完成的巡检点不需要重复检查

### 整改工单流程

**审核人不在**
- 问题类型配置多个审核人，按顺序自动流转
- 超过 24 小时未审核，自动升级到上级主管

**整改人拒绝**
- 整改人可以拒绝工单并说明原因
- 工单退回到审核人重新指派

**复核不通过**
- 工单退回到整改人重新整改
- 记录复核不通过的次数
- 超过 3 次不通过，升级到业务主管处理

**工单关闭**
- 提交人可以关闭自己提交的工单（仅限待审核状态）
- 业务主管可以强制关闭任何工单

### 数据统计

**统计维度**
- 时间维度：今日/本周/本月/本季度/本年/自定义时间范围
- 组织维度：全公司/指定部门/指定人员
- 设备维度：全部设备/指定分类/指定设备

**统计指标**
- 巡检完成率 = 已完成任务数 / 总任务数
- 巡检及时率 = 按时完成任务数 / 总任务数
- 设备异常率 = 发现异常的设备数 / 巡检设备总数
- 整改完成率 = 已完成整改数 / 总整改数
- 整改及时率 = 按时完成整改数 / 总整改数
- 整改超期率 = 超期整改数 / 总整改数
- 一次复核通过率 = 一次复核通过数 / 总整改数

**数据导出**
- 支持导出 Excel 格式
- 导出内容包含明细数据和统计汇总
- 导出权限控制：仅管理员和业务主管可导出
