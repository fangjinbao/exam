# 智能AI考试平台设计方案

## 一、项目概述

智能AI考试平台是一套以人工智能为核心能力的在线考试与认证系统，覆盖「出题 → 组卷 → 考试 → 阅卷 → 认证发证」全流程。平台通过AI辅助出题、智能组卷、自动阅卷提升命题与评卷效率，结合防作弊技术保障考试公平，服务于职业资格认证、企业培训考核、技能测评等场景。

核心价值：
- AI 深度赋能命题与阅卷，降低人工成本
- 多终端覆盖，适配不同考试与练习场景
- 全链路防作弊，保障考试严肃性与公平性
- 认证发证闭环，支撑资格认证业务

## 二、功能范围

### 终端构成
- 管理后台（Web）：全平台业务管理
- 监考端（Web）：实时监控与异常处置
- 网页考生端（Web）：在线考试与练习
- 桌面考试端（Electron）：强防作弊考试
- 移动端（Vue + Capacitor）：移动考试与练习

### 管理后台模块（13个）
1. 题库管理：题目、题型、难度、知识点分类维护；AI出题、AI文档导入出题、AI生成解析
2. 试卷管理：手动组卷、AI组卷、多维度抽题规则模板、试卷预览
3. 考试管理：选试卷/选题库两种模式创建考试、考生分配、防作弊策略配置
4. 阅卷中心：客观题自动判分、主观题AI阅卷、人工复核与评分修正、复核记录留痕
5. 资格认证：认证项目管理、报考审核、认证与考试关联
6. 监考中心：监考安排、异常事件处置、录像回放入口
7. 证书管理：证书模板、颁发、编号、有效期、在线查验
8. 联系管理：考生与机构联系人信息维护
9. 报表中心：考试统计、成绩分析、AI学情分析报告、通过率报表
10. 组织管理：组织架构、部门、考点
11. 用户管理：账号管理
12. 角色管理：多角色RBAC权限
13. 系统管理：参数配置、AI模型配置（切换/密钥/prompt模板）、字典、日志

### 考生端功能
- 登录、考试列表、在线答题（倒计时/自动保存/断线续考）、在线练习
- AI答疑、错题本、AI推题、成绩查询、证书查看
- 桌面端额外：强制全屏、切屏检测、快捷键屏蔽
- 移动端额外：人脸核身、拍照抓拍上传

### 监考端功能
- 多考生画面墙、实时异常告警、事件流、远程提醒/终止考试、录像回放

## 三、核心模块设计

### AI 抽象模块
统一 AI 服务接口，业务层不感知具体厂商。ProviderRegistry 适配多模型（通义/DeepSeek/智谱/OpenAI），PromptTemplate 管理出题/阅卷/解析模板，长任务走队列异步执行，含重试/限流/降级/调用日志。模型配置存数据库，后台可切换，密钥加密存储。

### AI 出题流程
选知识点+题型+难度+数量 → 拼装prompt → 调AI → 结构化解析 → 生成待审题目 → 人工审核入库。AI生成题必须经人工确认才入正式题库。

### 组卷与创建考试
- 选试卷模式：直接使用已组好的固定试卷
- 选题库模式：按【题型×难度×知识点】配置每类题量和每题分值，系统随机抽题
- AI组卷：给定总分/时长/知识点分布，AI推荐抽题方案
- 每个考生可按乱序策略生成不同卷面

### 阅卷流程
交卷后客观题即时自动判分；主观题进入AI阅卷队列，AI按评分标准打分并给评语；阅卷老师复核可修正分数，记录复核人/时间/修改前后分值；汇总总分后发布成绩。AI阅卷失败可转纯人工，复核修改全程留痕。

### 防作弊体系
防切屏检测、人脸核身+定时抓拍、音视频/屏幕监控、操作限制（禁复制粘贴/右键/多屏/强制全屏）、题目乱序。所有作弊事件写入考试行为日志，监考端实时展示，考后可回溯。

## 四、数据结构与流向

核心实体：组织/用户/角色/权限；知识点分类/题型/难度/题目；试卷/抽题规则；考试/考试考生；答卷/答题项/判分结果/AI阅卷结果/人工复核记录；认证项目/报考记录/证书；考试行为日志/AI调用日志/AI模型配置。

数据流：题库 → 试卷 → 考试 → 答卷 → 阅卷（客观判分+AI阅卷+人工复核）→ 成绩 → 证书；认证项目 → 报考 → 考试；防作弊/监考 → 考试行为日志；AI抽象模块 → AI调用日志。

## 五、边界情况处理

- 断线续考：答案实时自动保存，重连恢复进度和剩余时间
- 考试超时：自动交卷，未答题按0分
- AI服务不可用：出题降级人工录入，阅卷降级纯人工，调用失败重试+告警
- 并发交卷/批量阅卷：队列削峰
- 权限越权：所有接口RBAC校验，考生只能访问自己的考试
- 证书防伪：唯一编号+查验码，在线验真
- 防作弊误判：告警不直接判违规，保留证据供人工裁定

## 六、技术栈

- 前端（管理后台/监考端/网页考生端）：Vue 3 + TypeScript + Vite + Element Plus
- 桌面考试端：Electron
- 移动端：Vue 3 + Capacitor
- 后端：NestJS + Prisma（单体模块化，AI抽象为独立模块，监考走WebSocket+Redis）
- 数据库：MySQL；缓存/队列/实时：Redis
